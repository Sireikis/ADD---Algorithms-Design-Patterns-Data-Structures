Summary

Linked Lists consist of segments called Nodes, which may point to another Node ahead of it, behind it, or both. Nodes contain some data and together compose a data structure that represents a sequence. Multiple variations of Linked Lists exist, each varying what references the Nodes are allowed to have. Some variations:

  • Singly-Linked List: Each Node references the next node and the last node points to nil.
  • Doubly-Linked List: Each Node references the node before and after it, with the last node pointing to nil.
  • Circular-Linked List: Each Node references the next node and the last node points to the first node.
