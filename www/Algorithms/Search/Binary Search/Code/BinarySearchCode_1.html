Binary Search

Iterative Search()


            func binarySearch(arr: [Int], target: Int) -> Int? {
                var lowIndex = 0
                var highIndex = arr.count - 1
                
                while lowIndex <= highIndex {
                    let middleIndex = (lowIndex+highIndex) / 2
                    
                    if target == arr[middleIndex] {
                        return middleIndex
                    } else if target < arr[middleIndex] {
                        highIndex = middleIndex - 1
                    } else {
                        lowIndex = middleIndex + 1
                    }
                }
                return nil
            }

Pseudocode



Succinct


1: Determine the first and last index, called the low and high indices
2: Search fails if the low index is greater than the high index
3: Determine the middle index
4: If the target is equal to the element at the middle index, return the middle index
5: If the target is less than the element at the middle index, high index becomes middle index minus one
6: If the target is greater than the element at the middle index, low index becomes middle index plus one
7: Repeat 2-6 until search completes

Verbose


Binary Search expects an array and a target.

In a zero-based array,
Declare the first index the lowIndex
Declare the last index the highIndex
While the lowIndex is less than or equal to the highIndex,
    Declare the average of lowIndex and highIndex the middleIndex
    
    If the target is equal to the element at middleIndex,
        Return the middleIndex
    Else if the target is less than the element at middleIndex,
        highIndex gets the value of middleIndex minus one
    Otherwise,
        lowIndex gets the value of middleIndex plus one
        
Search failed, return nil
