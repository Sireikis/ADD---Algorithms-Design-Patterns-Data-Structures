Selection Sort

Iterative Sorted() Succinct


            func SelectionSorted(for arr: [Int], lastIndex: Int) -> [Int] {
                var a = arr
                
                for i in 0...lastIndex-1 {
                    var jMin = i
                    
                    for j in i+1...lastIndex {
                        if a[j] < a[jMin] {
                            jMin = j
                        }
                    }
                    if jMin != i {
                        a.swapAt(i, jMin)
                    }
                }
                return a
            }

Iterative Sorted() Verbose


            func SelectionSorted(for arr: [Int], lastIndex: Int) -> [Int] {
                var array = arr
                var minValueIndex: Int
                var tempValue: Int
                
                for minUnsortedIndex in 0..<lastIndex {
                    minValueIndex = minUnsortedIndex
                    
                    for unsortedIndex in minUnsortedIndex+1...lastIndex {
                        if array[unsortedIndex] < array[minValueIndex] {
                            minValueIndex = unsortedIndex
                        }
                    }
                    tempValue = array[minUnsortedIndex]
                    array[minUnsortedIndex] = array[minValueIndex]
                    array[minValueIndex] = tempValue
                }
                return array
            }

Pseudocode



Succinct


1: Set minValue and unsortedIndex to index 0
2: Iterate through the array to find the smallest element, starting at unsortedIndex + 1
3: Update minValue with its index
4: Swap the element at minValue with the element at unsortedIndex
5: Increment unsortedIndex by 1
6: Repeat 2-5 until sorted

Verbose


In a zero-base array,
For every index A from 0 up to but not including the last index,
    Declare index A the minValueIndex
    
    For every index B to the right of index A up to but not including the last index,
        If the element at index B is less than the element at the minValueIndex,
            The minValueIndex becomes index B
    
    Swap the element at index A with the element at the minValueIndex
Return the sorted array
