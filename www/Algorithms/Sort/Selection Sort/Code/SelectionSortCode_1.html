Selection Sort

Iterative Sorted()


            func selectionSorted(for arr: [Int], lastIndex: Int) -> [Int] {
                var array = arr
                var minValueIndex: Int
                
                for minUnsortedIndex in 0..<lastIndex {
                    minValueIndex = minUnsortedIndex
                    
                    for unsortedIndex in minUnsortedIndex+1...lastIndex {
                        if array[unsortedIndex] < array[minValueIndex] {
                            minValueIndex = unsortedIndex
                        }
                    }
                    array.swapAt(minUnsortedIndex, minValueIndex)
                }
                return array
            }

Pseudocode



Succinct


1: Set minValueIndex and minUnsortedIndex to index 0
2: Iterate through the array to find the smallest element, starting at minUnsortedIndex + 1, called unsortedIndex
3: Update minValueIndex to equal unsortedIndex if unsortedIndex contains a smaller element
4: Swap the element at minValueIndex with the element at minUnsortedIndex
5: Increment minUnsortedIndex by 1
6: Repeat 2-5 until sorted

Verbose


In a zero-base array,
For every index A from 0 up to but not including the last index,
    Declare index A the minValueIndex
    
    For every index B to the right of index A up to and including the last index,
        If the element at index B is less than the element at the minValueIndex,
            The minValueIndex becomes index B
    
    Swap the element at index A with the element at the minValueIndex
Return the sorted array
