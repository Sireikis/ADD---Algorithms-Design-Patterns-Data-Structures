Selection Sort

Big O

Time Space
Best Average Worst Worst
O(n2) Θ(n2) O(n2) O(1)

Summary

Selection Sort virtually splits a given array into sorted and unsorted sections. An outer loop, beginning with the index of the first element, designates the current iteration as the MinValueIndex. Another loop, starting with the index to the right of the current iteration, compares all elements to the element at the MinValueIndex. If an element is found to be smaller (assuming ascending order sort), that element's index becomes the value of MinValueIndex. Once the inner loop reaches the end of its iteration through the 'unsorted' portion of the array, the element at the MinValueIndex is swapped with the element at the index designated by the outer loop (MinValueIndex's original value, if it changed). This places the smallest element of the entire array at the beginning.

At the beginning and end of every iteration of the outer loop, indexes to the left of the current outer loop index are sorted. The positions of these sorted elements will not change as sorting continues.
