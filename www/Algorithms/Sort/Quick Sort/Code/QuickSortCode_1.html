Recursive Sort()


            func quickSort(array: inout [Int], start: Int, end: Int) {
                if start < end {
                    let partitionIndex = partition(array: &array, start: start, end: end)
                    
                    quickSort(array: &array, start: start, end: partitionIndex-1)
                    quickSort(array: &array, start: partitionIndex+1, end: end)
                }
            }

            func partition(array: inout [Int], start: Int, end: Int) -> Int {
                let pivot = array[end]
                var pIndex = start
                
                for index in start..<end {
                    if array[index] <= pivot {
                        array.swapAt(index, pIndex)
                        pIndex += 1
                    }
                }
                array.swapAt(pIndex, end)
                
                return pIndex
            }

Pseudocode



Succinct


1: If the given range has less than two elements, return the array
2: Choose an element, called the pivot, in the given range
3: Place elements less than the pivot to its left and larger than it to its right
4: Repeat steps 1-3, where the range comprises elements to the left and right of the pivot

Verbose


Quick Sort Function: Expects a reference to an array, starting index, and ending index.
In a zero-base array,
Given a reference to the array and the starting and ending index,
If the starting index is less than the ending index,
    Declare an index called the partitionIndex, returned by the Partition Function
    
    Recursively call the Quick Sort Function with a reference to the array, the starting index, and partitionIndex minus one
    Recursively call the Quick Sort Function with a reference to the array, the partitionIndex plus one, and ending index
    
    
Partition function: Expects a reference to an array, starting index, and ending index.
In a zero-base array,
Given a reference to the array and the starting and ending index,
Declare the element at the ending index the pivot
Declare the starting index the pIndex
    
For every index A from the starting index up to but not including the ending index,
    If the element at index A is less than or equal to the pivot,
        Swap the elements at index A and the pIndex
        Increment pIndex
    
Swap the elements at pIndex and the ending index.
Return the pIndex
