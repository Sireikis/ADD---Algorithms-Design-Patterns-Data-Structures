Insertion Sort

Iterative Sorted()


            func insertionSorted(for arr: [Int], lastIndex: Int) -> [Int] {
                var array = arr
                
                var key: Int
                var index: Int
                
                for unsortedIndex in 1...lastIndex {
                    key = array[unsortedIndex]
                    index = unsortedIndex - 1
                    
                    while index >= 0 && array[index] > key {
                        array[index+1] = array[index]
                        index -= 1
                    }
                    array[index+1] = key
                }
                return array
            }

Pseudocode



Succinct


1: Index 1's value is the key
2: Compare the key to all elements on its left
3: Shift all elements that are greater than the key to the right
4: Insert the key
5: Advance the index of the key by one
6: Repeat 2-5 until sorted

Verbose


In a zero-based array,
For every index A from 1 to the last index,
    Declare the current element at index A the key
    Declare that the index to the left of index A is index B
    
    While index B is at least 0 and the element at index B is greater than the key,
        Copy the element at index B over the element to its right
        Decrement index B
    
    The element to the right of index B gets the value of the key
Return the sorted array
